# Market Observer - 投資市場観測・助言ツール

## 概要

市場ニュースを分析し、投資インパクトスコアを算出する観測ツールです。

**重要**: このツールは投資判断を行いません。

- ❌ 売買指示
- ❌ 銘柄・数量の提案
- ❌ 断定的な将来予測
- ✅ 情報収集・構造化
- ✅ 定量評価・変化検知
- ✅ 判断材料の提示

## セットアップ

```bash
pip install -r requirements.txt
```

## 使い方

```bash
python main.py
```

対話形式でニュースを入力し、日次レポートを生成します。

## 構成

```
market-observer/
├── main.py              # メインエントリポイント
├── analyzer/
│   ├── classifier.py    # ニュース分類
│   └── scorer.py        # インパクトスコア算出
├── alert/
│   └── detector.py      # アラート判定
├── report/
│   └── generator.py     # レポート生成
├── data/
│   └── logs/            # ログ保存
└── config.py            # 設定
```

## 投資インパクトスコア

- 範囲: **-10 〜 +10（整数）**
- 評価軸:
  1. 株価への方向性
  2. 影響の強度（短期/中期/構造）
  3. 影響範囲（個別/業界/市場）
